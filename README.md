# jazdr.github.io

Personal homepage for AI, security, and computer science research/engineering work.

Local preview:

```bash
open index.html
```

Publish to GitHub Pages:

```bash
git init
git add index.html cv.html styles.css README.md assets/Sungbin_Im_CV.pdf
git commit -m "Create personal homepage"
git branch -M main
git remote add origin git@github.com:jazdr/jazdr.github.io.git
git push -u origin main
```

The site will be available at:

```text
https://jazdr.github.io
```
